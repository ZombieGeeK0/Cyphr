# Cyphr

`[-] Install in Linux:`

```shell
git clone https://github.com/ZombieGeeK0/Cyphr.git && cd Cyphr && sudo bash install.sh && python3 cyphr.py
```
`[-] Install on Windows:`

- Download and unzip the repository
- `cd Cyphr & python cyphr.py`
